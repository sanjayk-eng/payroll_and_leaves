## ğŸ“ Usage Tips

### For Employees
1. **View Leaves**: Hover over colored blocks for details
2. **Apply Leave**: Click "Apply Leave" button (top-right)
3. **Check Status**: Color indicates status (yellow = pending, green/blue/red = approved)
4. **Navigate**: Use arrow buttons or "Today" to navigate

### For Managers
1. **Team Overview**: Scroll to "Team Leave Timeline"
2. **Identify Gaps**: Look for overlapping leaves
3. **Plan Resources**: Use timeline to see coverage
4. **Quick Stats**: Check dashboard for pending approvals

### Visual Cues
- **Ring around day** = Today
- **Gray background** = Weekend
- **Rose background** = Holiday
- **Badge with number** = Multiple leaves
- **Gradient card** = Interactive element
